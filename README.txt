How to Run

Digit Classification

_______________________________________________________
Naive Bayes
python main.py --dataset digits --algorithm naive_bayes

Perceptron
python main.py --dataset digits --algorithm perceptron


_______________________________________________________
Face Detection


Naive Bayes
python main.py --dataset faces --algorithm naive_bayes

Perceptron
python main.py --dataset faces --algorithm perceptron
_______________________________________________________
Feature Options

You may optionally specify a feature type:

--feature-type pixels
--feature-type counting
--feature-type pixels+counting
_______________________________________________________
Default feature type: pixels
